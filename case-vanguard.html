Vanguard

Interactive Motion & Responsive Design

Background

Vanguard was a class project designed to strengthen my front-end development skills through the creation of a modern, responsive website built with HTML5 and CSS. The goal was to turn provided mockups and content into a polished webpage that adapted across devices while meeting coding standards.

Problem Statement

The challenge was to design and code a website that looked professional on every screen size while including several technical requirements—validated semantic HTML5, responsive typography, grid-based layout, transitions, and a styled form.

Process

I began by forking the Vanguard repository on GitHub and cloning it locally. Using semantic HTML elements such as <header>, <section>, and <footer>, I built a clean structure with proper heading hierarchy.

Styling was done using a mobile-first CSS approach. I applied grid layouts, custom transitions, and responsive breakpoints at 800px and 1024px. The project also required imported fonts, an SVG graphic, and a fully functional mobile menu. After testing across multiple screen sizes, I deployed the project using GitHub Pages.

Results

The finished Vanguard webpage performed smoothly across all devices. It validated with zero HTML/CSS errors, included subtle animation for improved interaction, and demonstrated a strong understanding of responsive layout principles.
