Wunderlust Travel

Accessible, Mobile-First Travel Website

Background

Wunderlust Travel was a class project focused on writing responsive, accessible CSS for a pre-coded travel website. My goal was to follow brand guidelines, mockups, and HTML structure while creating a layout that adapted smoothly across mobile, tablet, and desktop screens.

Problem Statement

Travel websites are often cluttered and visually overwhelming. This assignment required creating a design that remained readable, mobile-first, and accessible, following WCAG best practices and validated CSS. The challenge was to write clean, responsive CSS using only the provided HTML.

Process

I began by adding the required CSS Reset and global variables, then built the mobile layout first. At 600px and 900px breakpoints, I introduced column layouts, horizontal navigation, and expanded grids. I tested on multiple devices to match the provided mockups and maintain spacing, contrast, and hierarchy.

Following the brand guide, I applied the correct fonts, colors, and image handling rules, ensuring accessibility and readability. The CSS validated with zero errors or warnings.

Results

The final Wunderlust Travel website displayed correctly across all screen sizes, meeting all rubric requirements. The design remained clean, accessible, and visually consistent with the brand mockups.
